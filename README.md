## Sobre.

Este projeto foi desenvolvido pensando em como organizar o Css e estrutura-lo, usando metodologia BEM, pastas e arquivos individuais para cada elemento tornando fácil e ágil manutenção em determinado ponto. Evitando deixar um único Css contendo mais de 500 linhas, onde dificulta qualquer manutenção.
Nas classes foram utilizadas as propriedades em ordem alfabéticas para também facilitar manutenção nas propriedades.

Realizado através do curso da Alura com a Oracle.
